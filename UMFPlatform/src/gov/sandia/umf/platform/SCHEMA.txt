===

Part (some are compartments some are connections)
 - Children: PartWrapper[]
 - Parents: Part[]
 - CanConnect: ConnectionWrapper[]|Map<String, ConnectionWrapper>|Map<String, ConnectionWrapper[]>
   - Intended use, capabilities 
   - if !null: I'm a connection part!
   - "A" / "B" , "Dest" / "Source", "Left" / "Right
   - Can be OPTIONAL...
 - Refs: ReferenceGlob[]
 - Metadata: Map<String,String>
 - Eqs: Equation[]
   - A.q = 12
   - B.r = 13    (which ones of these are intended
   - C.x = 12     to be used as connected parts?)
   - D.y = 13
ConnectionWrapper (choose better name)
 - Target: Part
 - Alias: String
 - ArrowHead?
PartWrapper
 - Alias: String
   - how to refer to the child, not very useful for cxn
 - UI-Info:
   - layout: ???? (different structure for comp/conn)
 - Target: Part
 - Connects: Map<Alias:String, PartWrapper>  actual bindings
   - Null for a part wrapper wrapping a compartment
   - Map of a connection's alias to a comp part wrapper
Equation
 - Value: String (e.g. "V' = cos(x) @ when")
 - UI-Info:
   - Order: Int (e.g. 7, for ordering eqs in the equation table) 
 - Description: String
 - Refs: ReferenceGlob[]
 - Metadata: Map<String,String>
ReferenceGlob
 - Ref: Reference
 - Reason: String
Reference (delegate to external reference / bib entry)
 - Author: String
 - Title: String
 - Journal: String
 - Link: String
 - Attachment: Blob (Optional)
